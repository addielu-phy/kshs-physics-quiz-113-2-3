# 高二物理自學評量網頁

本網站將「高雄中學 113 學年度第二學期高二物理第三次段考」製作成可作答、可自動評分、可重複練習的自學評量網頁。

- 題數：24 題
- 配分：第 1-20 題單選，每題 4 分；第 21-24 題多選，每題 5 分，總分 100 分
- 題面：由使用者提供的 DOCX 匯出 PDF 後逐題裁圖，保留原始公式與圖形
- 答案與解析：依題面推導而成，建議教師複核後正式使用

## 本機預覽

```powershell
python -m http.server 8731 --directory .
```

然後開啟 http://localhost:8731

## 更換題目

編輯 `data.js` 中的 `QUESTIONS` 陣列即可。題目本體目前使用 `assets/questions/q01.png` 至 `q24.png` 的逐題裁圖。
